# Анализ и прогнозирование популярности музыкальных треков Spotif

## Описание проекта

Проект посвящен исследованию факторов, влияющих на популярность музыкальных треков на платформе Spotify. В ходе работы проведен полный цикл анализа данных: от первичного исследования (EDA) до построения и сравнения моделей машинного обучения.

### Основные задачи

- Провести разведочный анализ данных (EDA) датасета Spotify
- Выявить ключевые факторы, влияющие на популярность треков
- Создать новые признаки (feature engineering) для улучшения предсказаний
- Обучить и сравнить модели из разных семейств алгоритмов
- Сформулировать практические рекомендации

### Результаты

- Жанр является главным фактором популярности (корреляция 0.504)
- Аудио-характеристики сами по себе слабо предсказывают популярность
- Лучшая модель по качеству: Random Forest (R2 = 0.489)
- Лучшая модель по скорости: XGBoost (0.5 сек на фолд)

## Структура проекта
- dataset.csv # Исходный датасет (114 000 треков, 20 признаков)
- Untitled.ipynb # Jupyter Notebook с полным анализом 
- requirements.txt # Список зависимостей
- venv/ # Виртуальное окружение Python
- .ipynb_checkpoints/ # Резервные копии notebook'а

## Содержание Notebook

1. **Импорт библиотек** - загрузка всех необходимых модулей
2. **Загрузка и первичный осмотр данных** - изучение структуры датасета
3. **Обработка пропусков** - заполнение缺失ных значений
4. **Анализ целевой переменной popularity** - статистика и визуализация
5. **Анализ популярности по жанрам** - выявление лидеров и аутсайдеров
6. **Корреляция признаков с popularity** - поиск зависимостей
7. **Матрица корреляции между признаками** - выявление мультиколлинеарности
8. **Распределение ключевых признаков** - анализ loudness, explicit, danceability, instrumentalness
9. **Scatter plots** - визуализация связей с популярностью
10. **Feature Engineering** - создание 7 новых признаков
11. **Сравнение корреляции старых и новых признаков** - оценка эффективности
12. **Сохранение данных для моделирования** - подготовка финального набора
13. **Random Forest - важность признаков** - анализ вклада каждого признака
14. **Сравнение моделей (5-fold CV)** - тестирование Ridge, Random Forest, XGBoost, MLP
15. **Итоговое заключение** - выводы и рекомендации

## Установка и запуск

### 1. Клонирование/копирование проекта

Скопируйте все файлы проекта в локальную папку.

### 2. Создание виртуального окружения

- python -m venv venv
- venv\Scripts\activate

### 3. Установка зависимостей

- pip install -r requirements.txt

### 4. Запуск Jupyter Notebook

- jupyter notebook

## Ключевые выводы

1. Жанр — главный фактор популярности (корреляция 0.504)

2. Аудио-признаки слабо коррелируют с популярностью (|r| < 0.1)

3. Random Forest показывает лучшее качество (R2 = 0.489)

4. XGBoost — оптимальный выбор для продакшена (баланс качества и скорости)
